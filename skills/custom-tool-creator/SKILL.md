# Custom Tool Creator Skill

## Overview

This skill provides comprehensive guidance for building custom tools for Roo agents. It covers all supported formats: CommonJS, ES Modules, and TypeScript.

## Key Features

- Complete templates for all formats
- Security best practices
- Error handling patterns
- Testing strategies
- Deployment instructions
- Multi-tool support

## Usage

Copy the appropriate template from `assets/`, implement your logic, and follow the guidelines in `references/`.

## Documentation

See `fragments/` for detailed guides and `references/` for reference materials.
